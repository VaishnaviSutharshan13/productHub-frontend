# ProductHub Frontend

This is the frontend of the ProductHub CRUD application built using Next.js.

## Features

- View all products
- Add new product
- Edit existing product
- Delete product
- Responsive UI
- Connects with backend API

## Tech Stack

- Next.js (App Router)
- JavaScript
- Tailwind CSS
- Axios

## Installation

npm install

## Run the App

npm run dev

Open in browser:
http://localhost:3000

## API Configuration

Create a file:

.env.local

Add:

NEXT_PUBLIC_API_URL=http://localhost:5000

## Note

Make sure backend server is running before starting frontend.
